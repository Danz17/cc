# Agent Configurations

This folder contains agent and persona configurations for Claude Code.

## Structure

```
agents/
├── mastermind.md     # Orchestrator agent
├── reviewer.md       # Code review agent
├── designer.md       # UI/UX design agent
└── researcher.md     # Research agent
```

## Usage

Copy agent files to your `~/.claude/` or project `.claude/` directory.
