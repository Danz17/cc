# CCR Presets

Claude Code Router presets for different use cases.

## Available Presets

- **phenix-smart-routing** - Intelligent multi-provider routing
  - Gist: https://gist.github.com/Danz17/c72c0c6eaa4c65b52fabcbaae479e184

## Usage

```bash
# Import preset
ccr preset import <gist-url>

# Apply preset
ccr preset apply phenix-smart-routing
```
