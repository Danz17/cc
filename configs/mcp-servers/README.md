# MCP Servers Configuration

This folder contains MCP server configurations for Claude Code and related tools.

## Structure

```
mcp-servers/
├── claude.json       # Claude MCP config
├── obsidian.json     # Obsidian MCP config
└── github.json       # GitHub MCP config
```

## Usage

Copy relevant configs to your `~/.config/claude/` or tool-specific config directory.
